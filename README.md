# Mundo Cinema V2

Aplicação de catálogo de filmes com autenticação de usuários.

## Sobre

Este projeto é uma versão evoluída do [Mundo Cinema](https://github.com/ivelcorvo/mundo-cinema), desenvolvido anteriormente com Create React App. Esta nova versão foi construída com Vite e tecnologias mais modernas, refletindo minha evolução como desenvolvedor.

## Tecnologias

- Vite 8
- React 19
- TypeScript 6
- Tailwind CSS v4
- React Router v7
- Axios
- Firebase (Authentication + Hosting)

## Como rodar

```bash
# Clone o repositório
git clone https://github.com/ivelcorvo/mundo-cinema-v2.git

# Entre na pasta
cd mundo-cinema-v2

# Instale as dependências
npm install

# Rode o projeto
npm run dev
```

Acesse `http://localhost:5173`
